# Setup
> TailwindCSS Installation
```bash
pnpm install
```
`Or`
```bash
npm install
```

# Running Development
> TailwindCSS
```bash
pnpm dev
```
`Or`
```bash
npm run dev
```

# Running Production
```bash
pnpm build
```
`Or`
```bash
npm run build
```
